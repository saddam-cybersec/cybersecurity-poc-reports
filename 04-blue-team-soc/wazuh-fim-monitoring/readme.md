# Wazuh FIM Monitoring PoC

This project demonstrates File Integrity Monitoring (FIM) and alert detection using Wazuh in a controlled security monitoring lab.

## Focus Areas

- File change detection
- Alert monitoring
- Log analysis
- Security visibility

## Tools Used

- Wazuh Manager
- Wazuh Agent
- Wazuh Dashboard
- Linux
