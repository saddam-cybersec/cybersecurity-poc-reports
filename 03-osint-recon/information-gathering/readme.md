# Information Gathering PoC

This project demonstrates reconnaissance and OSINT techniques for target profiling using publicly available tools and data sources.

## Focus Areas

- Subdomain enumeration
- WHOIS lookup
- Technology fingerprinting
- Email and metadata discovery

## Tools Used

- Kali Linux
- theHarvester
- Amass
- Subfinder
- Sublist3r
- Whois
- Wappalyzer
- httpx
